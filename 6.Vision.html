Vision

Corto Plazo

Ser la aplicación con mayor servicio fitness de todo México brindando la mejor capacitación a nuestros usuarios con ayuda de nuestras funciones para darles planes de entrenamiento personalizados y dietas. Popularizar la aplicación integrando una interfaz agradable para los usuarios y utilizando métodos de publicidad para seguir atrayendo más usuarios.

Mediano Plazo

Alcanzar un desarrollo total de la aplicación integrando y mejorando el uso de los servicios integrados a nuestra aplicación, innovar funciones para que los usuarios se les facilite el uso de esta. Tener un número de 50 mil usuarios activos integrando funciones las cuales permiten que los usuarios les sea agradable y fácil de usar con ayuda de encuestas para así adaptarse a nuestros usuarios.

Largo Plazo

Extender la aplicación internacionalmente, creando las condiciones necesarias para operar en el mercado internacional. Expandir el mercado de nuestra empresa, creando mercancía como playeras, shorts, joggers, tops, botellas y suplementos alimenticios, esto para la obtención de ingresos extras.
